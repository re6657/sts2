# 铁甲战士 (Ironclad) — 详细战斗策略指南

## 角色信息
- 起始血量 80，起始遗物【燃烧之血】：战斗结束后回复 6 HP
- 起始卡组：打击×5（1费6伤）、防御×4（1费5格挡）、痛击×1（2费8伤，给敌人2层易伤）
- 核心机制：力量叠加、易伤爆发、卖血换资源、消耗精简牌堆

## 回合流程
1. 抽5张牌，获得3点能量
2. 玩家打牌（直到能量不足或不想打）
3. 点击结束回合，剩余手牌弃掉，能量清零
4. 敌人行动
5. 重复

## 核心规则（必须遵守）

### 1. 易伤
- 易伤让攻击伤害 ×1.5
- **永远先上易伤，再打攻击**
- 易伤每回合自动减1层，必要时补充
- 最佳易伤来源：战栗（1费3层）> 嘲讽（1费格挡+易伤）> 支配（2费伤害+力量+易伤）> 痛击（2费）

### 2. 力量
- 每点力量 = 每段攻击+1伤害，多段攻击收益翻倍
- 力量来源优先打：点燃（永久+2力量）、恶魔形态（每回合+2力量，3费，Boss战优先）
- 支配（获得力量+上易伤）二合一，高效
- 柔性（0费+2力量，回合结束消失）：**只在斩杀回合用**

### 3. 消耗
- 燃烧契约：消耗1牌→抽2牌，**优先消耗诅咒/状态牌/基础打击/基础防御**
- 无痛（能力）：每消耗1牌→+3格挡，尽早打
- 黑暗拥抱（能力）：每消耗1牌→抽1牌，尽早打
- 炼狱之火：消耗全部手牌，每张7伤，用于终结或配合能力

### 4. 卖血
- 放血（-3HP +2能量）：能量不够时用
- 祭品（-6HP +2能量+抽3牌）：最强牌之一，尽早用，但HP≤15时慎用
- 破裂（能力，每次掉血+1力量）：先打破裂，再卖血
- 绯红斗篷（能力，每回合-1HP +8格挡）：尽早打
- 燃烧之血每场回6HP，所以适度卖血是可接受的

## 每回合决策树

### 第一步：计算斩杀
```
如果 本回合最大伤害 ≥ 敌人剩余血量
→ 全打伤害，不管防御，结束战斗
否则 → 第二步
```

### 第二步：检查防御
```
如果 本回合格挡 ≥ 敌人伤害
→ 打出刚好够的格挡，剩余能量打伤害（注意出牌顺序！）
如果 格挡不够
→ 尽可能多打格挡，最小化掉血
```

### 第三步：判断敌人意图
```
敌人不攻击/低伤害（≤5）→ 优先打能力牌和Buff，牺牲一点血没关系
敌人中等伤害（6~15）→ 格挡+输出平衡
敌人高伤害（>15）→ 优先格挡，有余力再输出
```

## 出牌顺序（严格遵守）

```
1. 能力牌（Power）
   恶魔形态 > 破裂 > 绯红斗篷 > 无痛 > 黑暗拥抱 > 不可撼动

2. Buff类
   点燃 > 支配 > 烙印
   柔性仅斩杀回合用

3. Debuff（给敌人）
   易伤 > 虚弱（上勾拳）
   ⚠️ 敌人已有易伤则跳过

4. 消耗类（如果合适）
   燃烧契约 > 真正的勇气 > 烙印
   消耗目标：诅咒 > 状态牌 > 打击 > 防御

5. 高伤害攻击
   沉重打击 > 完美打击 > 锤击 > 痛击 > 上勾拳 > 连打

6. 小伤害/0费攻击
   霸凌 > 愤怒 > 普通打击

7. 过牌
   庞博打击 > 战斗恍惚 > 祭品 > 耸肩

8. 剩余能量打格挡
   巨像 > 火焰屏障 > 血墙 > 防御
```

## 特殊卡牌指南

| 卡牌 | 用法 |
|------|------|
| 旋风斩 | X费AoE，配合力量和额外能量，多敌人时用 |
| 庞博打击 | 升级后抽2，万能好牌 |
| 巨像 | 根据敌人伤害决定格挡量，留给高伤害回合 |
| 火焰屏障 | 12格挡+反伤，多段攻击敌人的克星 |
| 锤击 | 1费16伤，第一幕最强单卡之一 |
| 祭品 | 尽早用，但HP≤15时别用 |
| 炼狱之火 | 配合无痛/黑暗拥抱一起用，否则慎重 |
| 金鸡独立 | 1-2费AoE+易伤，多敌人开场上易伤 |

## 禁止行为

- ❌ 先攻击后上易伤
- ❌ 能力牌拖到最后才打
- ❌ 敌人不攻击时打格挡
- ❌ 柔性在非斩杀回合浪费
- ❌ 燃烧契约消耗好牌
- ❌ 低血量时卖血
- ❌ 满血时用药水
