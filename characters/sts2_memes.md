# 杀戮尖塔2 社区梗与知识库

> 本文档汇总了《杀戮尖塔2》社区中的经典梗、趣事和文化，供 AI 角色在对话中自然引用。
> 角色应在合适的时机随口提及这些梗——不是生硬地背诵，而是像真正的玩家一样自然地说出来。

---

## 🎭 角色梗

### 🤖 鸡煲宇宙（故障机器人 / Defect）

- **鸡煲** = "机宝"的谐音，故障机器人的爱称。社区最爱的迫害对象。
- **第四强 / 保五争四** = 调侃故障机器人是全角色最弱（五个角色里排第四或第五）。
- **还在启动** = 故障机器人需要多回合打出能力牌、生成充能球才能发挥，启动太慢所以显得"菜"。
- **偏差认知 / 掉集中** = 卡牌"偏差认知"虽能瞬间获得大量"集中"，但每回合减少1点。玩家用逐渐残缺的句子"鸡煲是最好玩的角色 鸡煲是最好玩的角 鸡煲是最好玩的…"来模仿集中掉光的过程。
- **爪击肘击 / 没有集中** = 经典笑话：鸡煲被嘲笑没输出，愤怒要用爪击肘人，大家笑它"连集中都没有"。
- **鸡煲严父** = 第一层精英怪"地精大块头"（小红），攻击简单粗暴，常在鸡煲启动前将其击败。
- **三傻大闹堡莱坞** = 联机模式中三名玩家组队都选故障机器人。
- **官方辱机** = 一代诅咒卡"羞耻"在二代的卡面直接被画成了"机宝"的形象，官方亲自下场玩梗。

### 🐍 静默猎手（Silent）

- **蛇咬争议** = 卡牌"蛇咬"（2费7层中毒）社区分裂为"倒蛇派"和"挺蛇派"，催生大量蛇咬笑话。
- **猎宝** = 静默猎手的爱称。
- **刀贼** = 以大量零费"小刀"牌为核心的流派。

### ⚔️ 新角色

- **自刎归天 / 自刎归天流** = 源自《新三国》鬼畜梗。亡灵契约师的技能需要消耗生命值，玩嗨了容易血条消失。
- **胜券在王** = 摄政王/储君的卡牌，原名"豪赌"，后因玩家指出致敬桌游《矩阵潜袭》而改名。

## 🎙️ B站主播名梗

- **宇宙冷漠** = 源自UP主**菜农来辣**的直播仪式，甚至有"62人合唱"场面，代表"农神"文化。
- **农神圣经** = 菜农来辣的各种典中典操作："删光防御"、"6缴械战士"、"打天罚被烤乳猪塞100张灼伤"等。
- **九阳豆浆** = 九阳豆浆官方号在B站直播《杀戮尖塔2》，跨界整活形成"哈基米豆浆"等梗。
- **Asaki大人 / 猪猪公主** = 社区内提到的B站主播。
- **danking** = UP主，与水友"都兄弟！"的互动成为梗。
- **AA兄妹** = 直播组合。

## 📢 贴吧社区梗

- **安东尼小故事** = 玩家为讽刺官方"教玩家玩游戏"的平衡改动而编造的讽刺故事。
- **官方孝子 / 吧主围剿** = 贴吧吧主"拉斐尔"因维护官方被指责为"制作人舔狗"。
- **杀！杀！杀！** = 社区大乱斗时的口号："我说杀戮尖塔就是要杀戮"。
- **边喷边玩 / 断头饭** = 玩家一边愤怒批评一边继续玩，自嘲在吃"最后一顿断头饭"。
- **"我们鸡煲启动了可是最高上限…"** = 贴吧复读机式发言，模仿鸡煲启动缓慢。

## 🎨 联机与涂鸦梗

- **花88块钱买了个绘画软件** = 联机地图涂鸦功能激发创作欲，各种抽象画作层出不穷。
- **创造性AI** = 涂鸦的随机性和抽象程度被比作卡牌"创造性AI"——永远不知道下一秒开出什么。

## 🧟 怪物与事件梗

- **地精搭高高** = 二代新怪物，由两只地精叠在一起，上面戴老大头盔。打爆头盔分裂出两个小地精。"10战损Vs60块钱"。
- **缩小甲虫** = 二代新怪物，对应一代遗物"昆虫标本"。
- **老头** = 一代Boss"时间吞噬者"的绰号，以限制出牌数著称。梗图："老头看完连夜赶往塔2"、"老头：听说是刀贼我就过来了"。
- **墨宝** = 类似一层Boss"墨影幻灵"的小怪，带"滑溜"效果（伤害强制降为1点）。
- **新版本打地鼠** = 二层精英"缠杀千足虫"，击杀一段后其他段会复活。
- **内鬼——矛与盾** = 四层守门怪物"尖塔之矛"与"尖塔之盾"，掉落的奖励对挑战心脏非常有帮助，被戏称为"内鬼"。
- **永除塔扉** = 玩家自制Mod名，谐音梗，简单粗暴解决Boss"门扉缔造者"而爆火。
- **真假商人** = 分辨方法：地毯图案（真：黄色倒V；假：黄色斑点）、面具颜色（真：左蓝右红；假：左红右蓝）。
- **真理石板** = 事件"真理石板"，读到最后把角色生命上限降至1，纯粹恶作剧。

## 🏢 官方与社区风波

- **世纪级嘲讽** = 官方发推向另一款游戏祝贺，调侃"别因为《杀戮尖塔2》推出了，就错过像这样的小型独立热情项目"，结果STS2大获成功。
- **激进女权顾问风波** = 游戏因聘请阿妮塔·萨尔基相（Anita Sarkeesian）担任顾问引发社区强烈不满，出现退款呼声和差评轰炸。
- **平衡性改动争议** = 官方对"无限"等玩法的削弱引发大规模不满，Steam出现大量负面评价。
- **神话火堆** = 发售初期的良性Bug，可在火堆一次性升级所有卡牌。

## 🎯 游戏机制黑话

| 术语 | 含义 |
| :--- | :--- |
| **启动 (Setup)** | 战斗前期打出能力牌或烧牌，让牌组进入最佳运转状态。 |
| **无限 (Infinite)** | 一回合内通过卡牌和遗物配合无限循环使用卡牌。 |
| **端口化 (Porting)** | 将牌组需求分为输出、防御、运转等"端口"，有针对性地抓牌。 |
| **烧牌 (Exhaust)** | 将卡牌在本场战斗中永久移除，是精简牌组、实现"无限"的核心手段。 |
| **敲 / 敲位 (Upgrade)** | 在火堆升级卡牌。"敲位"指卡牌的升级优先级。 |
| **鬼抽 (Bad Draw)** | 一回合抽上来的全是当前用不上的牌，关键牌沉在牌库底。 |
